# 术语对照表

用于统一 Android -> iOS 迁移中的关键术语表达，避免同一概念多种叫法。

维护规则
- 新增或重命名核心类时，必须同步更新本表。
- `xmnote/UIComponents` 下可复用 UI 组件必须纳入本表。
- `xmnote/Views/<Feature>/Components` 下页面私有子视图必须纳入本表（类别：`UI-页面私有`）。
- 页面壳层（`xmnote/Views/<Feature>/*View.swift`）建议标记为 `UI-页面壳层` 或 `UI-核心页面`。
- 非复用但页面核心功能组件，若进入白名单，也必须纳入本表。
- 提交前执行：`bash scripts/verify_glossary.sh && bash scripts/verify_ui_glossary_scope.sh && bash scripts/verify_view_component_boundaries.sh`。
